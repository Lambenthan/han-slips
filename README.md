# Han Slips · 长沙五一广场东汉简牍词汇表

本仓库收录长沙五一广场东汉简牍研究的注释词汇表整理成果，共 **66 条**词条，源自 6 篇学术论文。每条词条包含术语 + 简号 + 完整释义 + 古籍引证 + 论文页码。

附带交互式审查工具，可在浏览器中实现 PDF 原文与注释释义的左右双栏对照。

## 仓库结构

```

├── 學習材料/                 金标准范本（3 篇论文 + 标准格式 docx）

└── 全新材料/                 主交付（3 篇论文 + 46 条词条 + 审查工具）
```

## 审查工作站

每份材料目录下都有一个 `审查工作站.html`，双击用浏览器（推荐 Chrome）打开即可使用。

**功能：**

- 三篇论文 Tab 切换
- 左侧条目列表（带审查状态指示灯）
- 中间 PDF 原文（点击条目自动跳转到对应论文页）
- 右侧注释释义 + 个人审查笔记
- 进度跟踪 + 键盘快捷键 + Markdown 报告导出

**键盘快捷键：**

| 按键 | 功能 |
|---|---|
| `←` `→` | 切换条目 |
| `C` | 标记"已查" |
| `Q` | 标记"存疑" |
| `E` | 标记"错误" |
| `U` | 取消标记 |

**页码偏移：** 如果 PDF 跳转的页码与论文页码不一致，调整顶部"页码偏移"输入框（自动保存到 localStorage）。

详细说明见各子目录的 `README.md`。

## 词条格式

每个词条遵循统一格式：

```
术语A·术语B·术语C（简号）

物理描述（长×宽厘米） + 文字断句 + 人名标注 + 
[N]编号的逐条词义解释，含《说文解字》《汉书》《二年律令》等古籍引证。XXX-XXX頁
```

源数据保存在各子目录下的 `_work/<论文目录>/entries.md` 中，对应的合并版本为 `_work/combined.md`。

## 生成 docx

如果需要从 markdown 重新生成 docx，使用 pandoc：

```bash
cd 驗證材料   # 或 全新材料
pandoc _work/combined.md -o 驗證材料.docx --reference-doc=../學習材料/學習材料.docx
```

## 浏览器兼容性

- **推荐**：Chrome / Edge（内置 PDF 查看器对 `#page=N` 跳转支持完善）
- **不推荐**：Firefox（PDF.js 对 iframe `#page=N` 跳转响应不稳定）
- 如果 Chrome 打开后 PDF 区域空白，在地址栏输入 `chrome://settings/content/pdfDocuments` 关闭"下载 PDF 文件而非自动打开"
