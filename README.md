# 长沙五一广场东汉简牍词汇表整理

本仓库收录长沙五一广场东汉简牍研究的注释词汇整理成果，自六篇学术论文中辑录词条，按"一术语一词条"粒度排列，并附简号、释义、古籍引证及论文页码。仓库另备一份基于浏览器的对照审查工具，供使用者在阅读原始 PDF 时与释义一并对照。

## 一、概述

长沙五一广场东汉简牍出土于二〇一〇年，所记内容多为东汉中期临湘县廷的司法、行政、户籍、市易等基层档案，是研究东汉地方政治与法律史的重要资料。学界已对部分简文展开释读与考证，本仓库的目的在于：

1. 从已发表的若干研究论文中辑录术语释义，按学习材料范本之体例（一术语一词条）整理；
2. 以 Word 与 Markdown 双格式呈现，便于检索与转换；
3. 提供一项交互工具，将 PDF 原文与所辑释义并列展示，便于复核与修订。

## 二、仓库结构

```

├── 學習材料/       格式范本（三篇论文，定义词条排列体例）
```

## 四、文档体例

参考 `學習材料.docx` 体例，词条按下式排列：

```
术语（简号）

释义文字。论文引《说文》《汉书》《二年律令》等古籍考订。XXX頁
```

正文用宋体五号（10.5pt），段落两端对齐，无标题层级、无装饰，体例朴素，便于检阅。

「全新材料」因来源论文体量较大、释义层级较深，于文档末附"附录：简号档案"一栏，按简号汇总该简的物理形态（长×宽）、断句释读、人名标注及全部相关术语的逐条释义，用于查证主体术语条目的简文背景。

释义层面参引《说文解字》《汉书》《后汉书》《二年律令》《睡虎地秦简》《居延汉简》《奏谳书》《岳麓秦简》等传世与出土文献，遵循学界既有训诂体例。

## 五、对照审查工具

各材料目录下附 `审查工作站.html`，可在浏览器中直接打开（推荐 Chrome 或 Edge）。

### 5.1 界面构成

工具采用三栏式布局：顶端为论文切换与全局检索；左栏为词条索引；中栏内嵌 PDF 阅读器并随条目自动跳转至对应论文页；右栏显示释义全文，下方备审查状态按钮及个人笔记输入框。

### 5.2 主要功能

| 功能 | 说明 |
|---|---|
| 条目跳转 | 点击索引中任一词条，PDF 自动定位至对应论文页 |
| 状态标记 | 四级状态——未查、已查、存疑、错误 |
| 笔记记录 | 自动保存于浏览器 localStorage，关闭页面不丢失 |
| 全局检索 | 在标题、简号、释文中模糊匹配 |
| 报告导出 | 生成 Markdown 审查记录，按论文与状态分组 |

### 5.3 键盘操作

| 按键 | 功能 |
|:--:|---|
| ← / → | 上一条 / 下一条 |
| C | 标记"已查" |
| Q | 标记"存疑" |
| E | 标记"错误" |
| U | 撤销标记 |

### 5.4 页码偏移

PDF 文件首页未必对应论文起始页（多有期刊封面、目录在前）。若发现条目跳转不准，调整顶部"页码偏移"数值即可。计算式：

```
PDF 实际页 = 论文页码 − 偏移量
```

各论文默认偏移已按起始页推算并填入，使用者亦可手动校准。

## 六、技术备注

- **浏览器选择**：审查工作站依赖浏览器内置 PDF 查看器对 `#page=N` 跳转的支持。Chrome 与 Edge 表现最稳。Firefox 使用 PDF.js 渲染，部分版本对 iframe 中的 `#page=N` 响应不一致。
- **本地 PDF 加载**：若 Chrome 打开 HTML 后 PDF 区域空白，进入 `chrome://settings/content/pdfDocuments`，关闭"下载 PDF 文件而非自动在 Chrome 中打开"。
- **数据隔离**：审查状态与笔记保存于浏览器 localStorage，每浏览器、每电脑独立。如需与他人协作，请通过工具内置的"导出报告"功能生成 Markdown 记录共享。
