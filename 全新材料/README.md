## 三、审查工作站使用

### 3.1 打开方式

双击 `审查工作站.html`。推荐使用 Chrome 或 Edge 浏览器。

### 3.2 界面布局

工具采用三栏式：顶端工具栏（论文切换、检索、进度、导出）、左侧词条索引、中央 PDF 阅读区、右侧释义与笔记区。

### 3.3 检查流程

1. 于左侧索引中点击任一词条；
2. 中央 PDF 自动定位至对应论文页；
3. 右侧显示释义全文，对照阅读后选择状态（已查、存疑、错误）；
4. 在笔记区记录复核意见，输入即自动保存。

### 3.4 键盘快捷键

| 按键 | 功能 |
|:--:|---|
| ← / → | 上一条 / 下一条 |
| C | 已查 |
| Q | 存疑 |
| E | 错误 |
| U | 撤销标记 |

### 3.5 页码偏移

若 PDF 跳转页码与论文页码不符，调整顶端"页码偏移"。计算式：

```
PDF 实际页 = 论文页码 − 偏移量
```

默认偏移：

| 论文 | 偏移量 | 备注 |
|---|:--:|---|
| 选释一 | 296 | 论文起始页约为 297 |
| 壹释读二 | 324 | 论文起始页 325 |
| 县廷告劾 | 123 | 论文起始页 124 |

调整后自动保存于浏览器 localStorage，下次打开仍生效。

### 3.6 报告导出

工具栏"导出报告"按钮生成 Markdown 格式审查记录，按论文与状态分组，包含个人笔记，可复制至剪贴板或下载为 `.md` 文件。

## 四、技术备注

- **数据隔离**：审查状态与笔记保存于浏览器 localStorage，每浏览器、每电脑独立。如需共享，请使用"导出报告"功能。
- **保持文件结构**：HTML 通过相对路径引用 PDF。请勿移动或重命名 PDF 文件，否则无法加载。
- **Chrome PDF 设置**：若 PDF 区域显示为空白，进入 `chrome://settings/content/pdfDocuments`，关闭"下载 PDF 文件而非自动打开"。
- **修改源数据**：编辑 `_work/<论文>/entries.md` 后，可用 pandoc 重新合并生成 docx：

  ```bash
  cat _work/选释注释译一/entries.md \
      _work/壹释读二/entries.md \
      _work/东汉县廷的告与劾/entries.md \
    > _work/combined.md
  pandoc _work/combined.md -o 全新材料.docx \
      --reference-doc=../學習材料/學習材料.docx
  ```
