# 示範：多論文聚合語詞彙釋

本目錄展示 `bamboo-slips-lexical-glossary` skill 的完整 pipeline：從多篇論文卡片到符合徐碩硕士論文範式的 Word 文檔。

## 文件清單

| 文件 | 說明 |
|---|---|
| `papers_codebook.json` | 論文簡稱表（含 14 位學者 + 8 個出版本） |
| `terms_raw.jsonl` | 語詞原始卡片（30 條，來自 6 篇論文） |
| `glossary.json` | 聚合後的辭條（20 條） |
| `glossary.md` | Markdown 渲染（可讀性最好） |
| `示範·語詞彙釋.docx` | 學術範式 Word 成稿 |

## 跑通完整 pipeline

```bash
# 1. 安裝依賴
pip install pypinyin
npm install -g docx

# 2. 聚合卡片為辭條
python ../scripts/build_glossary.py \
    --codebook papers_codebook.json \
    --raw terms_raw.jsonl \
    --out glossary.json \
    --md glossary.md

# 3. 渲染為 docx
NODE_PATH=$(npm root -g) node ../scripts/render_docx.js \
    --glossary glossary.json \
    --out 示範·語詞彙釋.docx \
    --title "示範·語詞彙釋"
```

## 範式驗證

本示範產出的 20 條辭條完整體現徐碩方法論的核心：**跨論文聚合**。

### 示例 1：[案] — 兩位學者解讀聚合

```
[案]

編號：
《壹》257   木牘  CWJ1③：71-26
            木牘  2010CWJ1③：71-26

【姚遠2022A】 通按，之下為說明內容，因此可以用冒號。（301-304 頁）

【林琦婧2022】 與劾相關的文書有其特殊格式：標誌性詞語是開頭的"案"和結尾處的
"敢言之"等詞句。木牘以"案：都鄉某里大男馬胡、南鄉不處里區馮，皆坐"開篇，
再列舉涉案人士的居住地與身份，敘述案情，交代被劾者罪名，末以舉劾者署名加
"敢言之"作結。（130-132 頁）
```

兩位學者對同一個"案"字的不同角度解讀（姚遠從訓詁角度，林琦婧從文書格式角度）被自動聚合到同一辭條下。

### 示例 2：[自言] — 三位學者解讀聚合

```
[自言]

編號：
木牘 CWJ1③：325-2-9 / 325-5-21
《壹》  竹簡 2010CWJ1①：25-23
       木牘 CWJ1①：92

【姚遠2021】 ...（329-332 頁）
【姚遠2022A】 ...（320-322 頁）
【林琦婧2022】 ...（127-129 頁）
```

三條解讀按學者首次發表時間自動排序。

### 示例 3：[邸土] — 兩位學者對同一字的不同見解

```
[邸土]

【姚遠2022A】 邸土：奴婢人名。學界對此案例的討論內容較多，主要圍繞免、邸土
是兩個奴婢的名字還是放免邸土的問題。

【文霞2023】 免、邸土：陳偉先生認為應該是放免邸土，"何"把散用錢歸為己有。
整理小組則將"免、邸土"解釋為兩個奴婢。筆者認同陳偉先生的看法。《說文解字》
將"免"解釋為會意字，意為"兔逸也，從免不見足"，段玉裁引申為"凡逃逸者皆謂之免
，假借為袒免"。此處應是將邸土免為庶人的意思。（76-77 頁）
```

學術爭論被忠實保留，讀者可以對照兩位學者的不同看法。

## 從本示範到 506 頁完整版

本示範僅含 20 條辭條（從 6 篇論文聚合）。若要復刻徐碩 506 頁的完整論文：

1. **擴大論文集** — 收集約 200 篇圍繞同一批簡牍的論文（徐碩參考文獻含 200 篇）
2. **逐篇抽卡** — 對每篇論文用 Read PDF 多模態閱讀，識別其中討論的所有語詞
3. **建立簡稱表** — 為每位學者及每年的論文建立統一簡稱（重名年份用 A/B/C 區分）
4. **批量聚合** — 跑 `build_glossary.py`
5. **撰寫緒論** — 寫 20-30 頁的緒論（出土公布情況、簡牍性質、專題研究綜述）
6. **撰寫附錄** — 辭條索引、簡號對照表、地名整理表等
7. **生成 docx** — 跑 `render_docx.js`

每篇論文約 1-2 小時的精讀時間，200 篇論文需要 300-400 小時。建議多人協作或分批分階段執行。

## 進一步擴展方向

- **整合 Read PDF 多模態管線**：可結合 Claude Code 的 PDF 多模態能力，半自動抽取論文中討論的語詞
- **辭條索引自動生成**：附錄一的索引可從 glossary.json 自動生成
- **新舊簡號對照表**：附錄二可從 codebook 中的簡號自動聚合
- **地名整理表**：附錄五可通過詞性標注自動識別所有地名辭條
