收录长沙五一广场东汉简牍研究三篇论文，按学习材料体例整理为词条文档，并附 PDF 原文与对照审查工具。本目录作为词条提取流程的验证集。

## 二、文档结构

成稿《驗證材料.docx》采用学习材料体例，按"一术语一词条"粒度排列，共 **20 条**。每条结构：`术语（简号）` + 释义。释义聚焦于该术语本身的解读及相关古籍引证。

验证材料的三篇论文均为篇幅较短的训诂讨论，原条目即为术语级，无须拆分为附录档案，故本文档仅含主体部分。

## 四、审查工作站使用

### 4.1 打开方式

双击 `审查工作站.html`，浏览器自动打开。**推荐使用 Chrome 或 Edge**。

### 4.2 界面布局

工具采用三栏式：顶端工具栏（论文切换、检索、进度、导出）、左侧词条索引、中央 PDF 阅读区、右侧释义与笔记区。

### 4.3 检查流程

1. 于左侧索引中点击任一词条；
2. 中央 PDF 自动定位至对应论文页；
3. 右侧显示释义全文，对照阅读后选择状态（已查、存疑、错误）；
4. 在笔记区记录复核意见，输入即自动保存。

### 4.4 键盘快捷键

| 按键 | 功能 |
|:--:|---|
| ← / → | 上一条 / 下一条 |
| C | 已查 |
| Q | 存疑 |
| E | 错误 |
| U | 撤销标记 |

### 4.5 页码偏移

若 PDF 跳转页码与论文页码不符，调整顶端"页码偏移"。计算式：

```
PDF 实际页 = 论文页码 − 偏移量
```

默认偏移：

| 论文 | 偏移量 | 备注 |
|---|:--:|---|
| 元遗产案 | 75 | 论文起始页 76 |
| 王皮木牍 | 12 | 论文起始页 13 |
| 札記二則 | 219 | 论文起始页 220 |

### 4.6 报告导出

工具栏"导出报告"按钮生成 Markdown 格式审查记录，按论文与状态分组，包含个人笔记，可复制至剪贴板或下载为 `.md` 文件。

## 五、技术备注

- **数据隔离**：审查状态与笔记保存于浏览器 localStorage，每浏览器、每电脑独立。如需共享，请使用"导出报告"功能。
- **保持文件结构**：HTML 通过相对路径引用 PDF。请勿移动或重命名 PDF 文件，否则无法加载。
- **Chrome PDF 设置**：若 PDF 区域显示为空白，进入 `chrome://settings/content/pdfDocuments`，关闭"下载 PDF 文件而非自动打开"。
