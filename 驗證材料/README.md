## 审查工作站使用方法

### 打开方式

双击 `审查工作站.html`，浏览器会自动打开。**推荐使用 Chrome 或 Edge**。

### 界面布局

- **顶部栏**：三篇论文 Tab 切换 / 审查进度 / 搜索框 / 页码偏移调整 / 导出按钮
- **左侧栏**：当前论文的全部条目列表（带状态指示灯）
- **中间区**：PDF 原文（点击条目自动跳转到对应页）
- **右侧区**：注释释义 + 审查状态按钮 + 个人笔记输入框

### 检查流程

1. 点击左侧任一条目
2. 中间 PDF 自动跳转到对应页码
3. 右侧显示该条目的完整注释
4. 对照检查后，点击底部状态按钮（已查 / 存疑 / 错误）
5. 必要时在笔记框记录问题（自动保存）

### 键盘快捷键

- `←` `→` 切换条目
- `C` 标记"已查"
- `Q` 标记"存疑"
- `E` 标记"错误"
- `U` 取消标记

### 页码偏移

如果点击条目后 PDF 跳转的页数与论文页码不一致，调整顶部"页码偏移"。

**规则**：PDF 第 N 页 = 论文页码 − 偏移量

默认偏移（如需调整可手动改）：
- 元遗产案：75（论文起始页 76）
- 王皮木牍：12（论文起始页 13）
- 札記二則：219（论文起始页 220）

调整后自动保存，下次打开仍生效。

### 导出审查报告

点击右上角"导出报告"，可生成 Markdown 格式的审查记录（按论文 + 状态分组，含个人备注），可复制到剪贴板或下载为 `.md` 文件。

## 注意事项

1. **数据保存位置**：检查状态和笔记保存在浏览器的 localStorage 中，每个浏览器/电脑独立。如果你在 Chrome 标记了"已查"，换到 Safari 打开会重新从空白开始。要分享审查结果，请用"导出报告"。

2. **保持文件结构**：HTML 通过相对路径引用 PDF 文件。请不要把 PDF 移到其他位置或重命名，否则 PDF 无法加载。

3. **Chrome PDF 设置**：如果点击条目后 PDF 区域空白，检查 Chrome 的 PDF 设置：
   - 地址栏输入 `chrome://settings/content/pdfDocuments`
   - 关闭"下载 PDF 文件而非自动在 Chrome 中打开"

4. **Firefox 兼容性**：Firefox 使用 PDF.js 渲染，#page= 跳转有时不准确。推荐使用 Chrome。
